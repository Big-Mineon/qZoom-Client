# qZoom-Client README
qZoom is a video conferencing application made for Linux. It theoretically supports up to 256 people in the same room.. your server may say otherwise. 

# Dependencies
* `Qt 5.*`
* `libavformat`
* `libavcodec`
* `libavutil`
* `libswresample`
* `libavdevice`
* `libavfilter`
* `libswscale`
* `libswresample`
* `libao`
